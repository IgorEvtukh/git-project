# Получение текущего баланса пользователя

Протокол позволяет мобильному приложению запросить актуальный баланс игрока, включая доступные средства и бонусный счет.

* **URL метода:** `/api/v1/user/balance`
* **HTTP-метод:** `GET`
* **Авторизация:** Требуется (Токен пользователя)

### Успешный ответ (Response 200 OK)

Если запрос прошёл успешно, сервер возвращает данные в формате JSON:

```json
{
  "status": "success",
  "data": {
    "user_id": 19196,
    "currency": "RUB",
    "balance_available": 1500.00,
    "balance_bonus": 500.00,
    "balance_total": 2000.00
  }
}
```

### Описание параметров ответа (внутри объекта data)

| Параметр | Тип данных | Обязательный | Описание |
| :--- | :--- | :--- | :--- |
| `user_id` | Integer | Да | Уникальный идентификатор пользователя в системе |
| `currency` | String | Да | Трехзначный код валюты счета (например, RUB, USD) |
| `balance_available` | Float | Да | Сумма доступных средств для совершения ставок |
| `balance_bonus` | Float | Нет | Сумма бонусных средств на счету игрока |
| `balance_total` | Float | Да | Общая сумма счета (`balance_available` + `balance_bonus`) |
