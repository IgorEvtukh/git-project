# Создание новой ставки (Прием купона)

Метод регистрирует новую ставку пользователя в системе, проверяет доступность баланса и резервирует средства под купон.

* URL метода: /api/v1/bets/place
* HTTP-метод: POST
* Авторизация: Требуется (Токен пользователя)

### Тело запроса (Request Body)

Приложение отправляет на сервер JSON-пакет со следующими параметрами ставки:

```json
{
  "match_id": 85732,
  "outcome_id": "home_win",
  "coefficient": 2.10,
  "bet_amount": 1000.00
}
```
