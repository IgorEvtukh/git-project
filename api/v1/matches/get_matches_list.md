# Получение списка активных матчей

Метод возвращает список всех доступных для ставок матчей, сгруппированных по видам спорта и турнирам, с базовыми коэффициентами (П1, Ничья, П2).

* URL метода: /api/v1/matches/list
* HTTP-метод: GET
* Авторизация: Не требуется

### Параметры запроса (Query Parameters)

Эти параметры передаются в конце URL-строки для фильтрации выдачи:

| Параметр | Тип данных | Обязательный | Описание |
| :--- | :--- | :--- | :--- |
| sport | String | Нет | Фильтр по виду спорта (например: football, hockey, basketball). Если не указан — выводятся все виды спорта. |
| type | String | Нет | Тип линии: prematch (будущие события) или live (матчи в реальном времени). Если не указан — возвращаются оба типа. |

### Успешный ответ (Response 200 OK)

Сервер возвращает структурированное дерево активных событий, сгруппированных по видам спорта и лигам:

```json
{
  "status": "success",
  "data": {
    "sports": [
      {
        "sport_id": "football",
        "sport_name": "Футбол",
        "leagues": [
          {
            "league_id": 101,
            "league_name": "Лига Чемпионов УЕФА",
            "matches": [
              {
                "match_id": 85732,
                "match_type": "live",
                "home_team": "Фенербахче",
                "away_team": "Галатасарай",
                "start_time": "2026-06-03T20:00:00Z",
                "current_score": "2:1",
                "main_odds": {
                  "W1": 1.85,
                  "X": 3.40,
                  "W2": 4.20
                }
              }
            ]
          }
        ]
      }
    ]
  }
}
```

### Описание параметров успешного ответа

| Параметр | Тип данных | Обязательный | Описание |
| :--- | :--- | :--- | :--- |
| sport_id | String | Да | Уникальный текстовый идентификатор вида спорта (football, hockey) |
| sport_name | String | Да | Локализованное название вида спорта для отображения на экране |
| league_id | Integer | Да | Идентификатор турнира / лиги |
| league_name | String | Да | Название турнира (например, Лига Чемпионов УЕФА) |
| match_id | Integer | Да | Уникальный идентификатор матча в системе |
| match_type | String | Да | Текущий статус игры: prematch или live |
| home_team | String | Да | Название команды хозяев |
| away_team | String | Да | Название команды гостей |
| start_time | String (ISO 8601) | Да | Дата и время начала матча в формате UTC |
| current_score | String | Нет | Текущий счет матча. Передается только для событий с типом live |
| main_odds | Object | Да | Объект с базовыми коэффициентами на исход основного времени |
| W1 | Float / Null | Да | Коэффициент на победу хозяев (П1). Может быть null, если исход заблокирован |
| X | Float / Null | Да | Коэффициент на ничью. Может быть null для видов спорта, где ничьих нет |
| W2 | Float / Null | Да | Коэффициент на победу гостей (П2). Может быть null, если исход заблокирован |
