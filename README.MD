# Betting Shop API (MVP)

Спецификация и архитектура серверного API для букмекерского мобильного приложения.

## Структура проекта

* /api/v1/bets/ — Модуль ставок (оформление купонов, история пари).
* /api/v1/matches/ — Модуль спортивной линии (список матчей, детальная роспись исходов).
* /api/v1/user/ — Модуль профиля (баланс, пополнение счета).

## Стек технологий
* Архитектурный стиль: REST API
* Формат данных: JSON
* Формат дат: ISO 8601 (UTC)
