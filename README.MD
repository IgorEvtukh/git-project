# Betting Shop API (MVP)

Спецификация и архитектура серверного API для букмекерского мобильного приложения.

## Структура проекта

* /api/v1/bets/ — Модуль ставок (оформление купонов, история пари).
* /api/v1/matches/ — Модуль спортивной линии (список матчей, детальная роспись исходов).
* /api/v1/user/ — Модуль профиля (баланс, пополнение счета).

## Стек технологий
* Архитектурный стиль: REST API
* Формат данных: JSON
* Формат дат: ISO 8601 (UTC)

## Как пользоваться документацией

Все методы API разделены по функциональным модулям в папке `/api/v1/` . 
Каждый метод описан в отдельном `.md` файле и содержит:
1. HTTP-метод, URL и требования к авторизации.
2. Структуру и параметры тела запроса (Request Body).
3. Пример успешного JSON-ответа (`200 OK`) с подробным описанием каждого поля.
